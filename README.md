This is an interactive console-based C++ program that allows users to build a custom pizza by choosing crust, sauce, and ingredients from a menu, along with specific quantities. It was developed as a group project in an introductory C++ programming course.

---

## Features

- Select crust type (e.g., regular, gluten-free)
- Choose sauce and its amount
- Browse a list of 12 customizable toppings
- Pick individual portion sizes per ingredient
- See a final summary of your custom pizza
- Menu-driven interface using letters and clear prompts

---

## Technologies Used

- C++
- Standard libraries: `iostream`, `vector`, `string`, `iomanip`
- Console-based UI
